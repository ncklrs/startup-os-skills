## Complete Storyboard Script Template

## Example: Abbreviated 30s Social Ad

For quick reference, here's a condensed example:

---

# "Fast Deployment" Social Ad

**Type**: Social Ad
**Duration**: 30 seconds
**Objective**: Trial signups
**Framework**: Hook-CTA

## Story Arc
Hook (0-5s): Problem question
Demo (5-22s): Solution in action
CTA (22-30s): Clear next step

---

### Scene 1: Hook (0-5s)

**Narrative Purpose**: Immediate pain point identification

**On-Screen Copy**:
```
Deployments taking hours?
```

**Visual**:
- Background: Dark, code editor with loading spinners
- Main: Progress bar stuck at 43%, clock showing 2 hours elapsed
- Person waiting, frustrated

**Psychology**: Instant recognition - "that's my problem"

**Motion Notes**: Fast pace, build tension

**Transition**: Dark frustration → light relief

---

### Scene 2: Solution Demo (5-22s, Duration: 17s)

**Narrative Purpose**: Show product solving problem

**On-Screen Copy**:
```
Deploy in 90 seconds with Shipfast
```

**Visual**:
- Background: Clean, light interface
- Main: Simple command line: `shipfast deploy`
- Progress: 0s → 90s → "Deployed ✓"
- Split screen: Code → Live site

**Psychology**: "It's that simple?" + speed proof

**Sales Element**: Specific time claim (90s) with visual proof

**Motion Notes**: Medium pace, let demo breathe

**Transition**: Success moment → credibility

---

### Scene 3: Social Proof (22-26s, Duration: 4s)

**On-Screen Copy**:
```
Join 8,000 developers shipping faster
```

**Visual**: Logos: Vercel, Railway, Fly.io

**Psychology**: FOMO + credibility

---

### Scene 4: CTA (26-30s, Duration: 4s)

**On-Screen Copy**:
```
Start deploying faster
Free for 14 days • No credit card
```

**Visual**: Large CTA button, product in background

**Psychology**: Clear action, friction removed

---

**Assets Needed**:
- Logo: Shipfast SVG
- Product: Terminal screenshot
- Logos: Vercel, Railway, Fly.io (monochrome)

**Style**: Clean, technical, fast-paced
